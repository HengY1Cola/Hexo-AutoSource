## 自动部署

参照了B站Up(objtube的卢克儿)的代码实现了自动打包部署到main分支下

达到了更新源文件从而更新main分支，服务器直接拉取main分支最新文件即可

点击右侧进行查看效果👉 https://github.com/HengY1Sky/Hexo-AutoSource/tree/main

部署源码在`.github/workflows/deploy.yml`中, 分享拿走！
